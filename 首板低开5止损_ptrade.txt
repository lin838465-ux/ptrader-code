# 首板低开5止损策略 - PTrade 版本（纯文本规格）

## 一、策略概述
- 交易品种：A股（不含ST、科创、北交所）
- 周期频率：日线（定时任务）
- 核心逻辑：昨日首次涨停 → 今日低开3%-4% → 买入，止损-5%，11:28/14:50止盈清仓

## 二、信号规则

### 买入信号
1. 昨日收盘价 = 昨日涨停价（首板）
2. 往前数两根K线均未涨停（排除连续涨停）
3. 近60日最低价位置 ≤ 50%（相对低位）
4. 上市时间 > 250天（非新股）
5. 今日开盘价 / 昨日收盘价 ∈ [0.96, 0.97]（低开3%-4%）
6. 非ST、非停牌

### 卖出信号
- 止损：当前价 ≤ 成本价 × 0.95
- 止盈（11:28）：成本价 < 当前价 < 涨停价
- 清仓（14:50）：当前价 < 涨停价

## 三、参数配置
| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| 无外部可配置项 | - | 逻辑内嵌 |

## 四、时间轴
| 时间 | 动作 |
|------|------|
| 09:30 | 买入检查 |
| 09:35 | 卖出检查 |
| 11:28 | 止盈卖出 |
| 14:50 | 清仓卖出 |

## 五、参考对比（对照 etf23）
- 与 etf23 相同点：T+1检查、防重复下单机制、涨跌停检查、实盘回调
- 与 etf23 差异点：etf23是ETF轮动（ trailing stop），本策略是首板低开模式；本策略用ST过滤+新股过滤

## 六、PTrade API 差异点（JoinQuant → PTrade）
| JoinQuant | PTrade |
|-----------|--------|
| context.current_dt | context.blotter.current_dt |
| context.previous_date | _get_previous_trading_day(context) |
| pos.closeable_amount | pos.enable_amount |
| order_value(code, cash) | order(code, shares, limit_price=price) |
| set_option/log.set_level | 删除 |

## 七、文件结构
- 状态持久化：无（不复用昨日数据，每日独立运行）
- 配置读取：无外部配置
